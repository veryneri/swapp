# SWAPP

Simple [React Native](https://facebook.github.io/react-native/) application to display Star Wars Movies.

## Quick start

Get up and running with SWAPP:

### Install dependencies:
```
npm install
```

### Running SWAPP:

- Android:
```
npm run android
```

- iOS:
```
npm run ios
```
or open the `ios/Swapp.xcodeproj` project in Xcode.
